# Tecnotics Platform

Plataforma web oficial de Tecnotics. La version `0.1.0` prepara la base del ecosistema digital: sitio corporativo, servicios, SEO y estructura futura para ecommerce, CRM, ordenes de trabajo, inventario, dashboard, IA y Laboratorio Arcade.

## Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Supabase / PostgreSQL preparado para etapas siguientes
- Vercel como hosting objetivo

## Comandos

```bash
npm install --legacy-peer-deps
npm run dev
npm run build
```

## Estructura inicial

- `src/app`: rutas publicas de la plataforma.
- `src/components`: componentes reutilizables.
- `src/lib`: configuracion y contenido compartido.
- `public/images`: assets visuales del sitio.
- `docs`: documentacion maestra, arquitectura y roadmap.
